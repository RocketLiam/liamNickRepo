Object-Oriented Coding

In an Object-Oriented programing lanague such as java you have this idea of classes. classes are able to store data and the functions in one place. Older lanagues would have had these separated. An object is a class that has been built and running in memory.

Variables

Variables is the name given to the data in classes. In java these to be given a type so that the code knows what to do with it. Along with making it clear what data that stores for people looking at your code.

Methods

These are the functions that chage the variables (data) in the objects. Methods can take an input, and can provide and output. Allowing you to work with and change anything into anything. USE WISELY
