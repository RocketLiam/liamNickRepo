Unit 1

Object-Oriented Programming

Overview

Unit 1 explains the basics of Java and goes over what Object Oriented Programming is.

Object-Oriented Programming

Object-Oriented Programming uses the idea of classes and objects. A class is able to store data and functions. An object is an instance of a class with its own local variables. For example, you could have a class called Person, with the variables of name and age. You could then create different objects of the Person class, with different values for those variables. You could create a Person with the name Dave and the age 45, and then create a different Person with the name Bob and the age of 27. You could then access the data from each object separately.

Variables

Variable is the name given to the data stored in classes. In java, you give variables a type so that the computer knows what to do with it. Different types of variables contain different things. For example, int's contain integers, and booleans contain true or false. There are many different types of variables, used for different things.

Methods

Methods, also known as functions, perform actions by changing variables (data) within objects. Methods can take inputs and give outputs, or it could do neither. These allow you to manipulate and change any data that you have. So use them wisely.
