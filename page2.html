Constructors

There are two types of constructors. These are No-Argument Constructors and Paramaterized Constructors. No-Argument Constructors set the default variables when creating an object from a class. Paramaterized Constructors take input values when instantiating a class to create an object. These values are then transferred to local variables in the newly created object.

Mutators

Mutators, also known as Setters, allow you to change the variables stored in an object. They are written as a method with a return type of void, as you are trying to change a variable, not access one. The method typically is named by using Set, followed by the variable name. For example, if you are trying to change a String called name, you would name the method SetName.

Accessors

Accessors, also known as Getters, allow you to read the variables set by the Constructor or Mutator.
