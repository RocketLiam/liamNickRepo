Unit 4

Conditions and Logic

Overview

This unit covers how to use the math class in java. Along with what static variables and methods are.

The Keyword Static

The keyword static cause the method or variable it is given to becomes tied to the class not the object. So if a variable has the static keyword across all object created from that class will read the same value. Simlar with methods they will run the same over all object created from that class.

Math

Allows easy access to a number of mathematical functions. Such as sqaure root, absolute value, pow and many more. Math is a static method so you don't need to instantiate an object of it. You just need to do Math.blah.

Casting

The action of treating a piece of memory as another type of memory. Like treating a double as an int. So can do operations as one type of memory then be read as other kind of memory.
