Unit 3

Arrays and Algorithms

Overview

You get into how to better store variables in arrays and how to access them when they are stored in arrays.

1D arrays

An array is a way of storing variables of the same type neatly. Insted of having to change each variable by name you can just create an array of them and store, read, and change them in the array. An array in this case just discribes how the computer is storing the memory. Which is sequently, so each variable will be one after the other. The 1D simply refers to the dimension of the array. Which is the amount of arrays in arrays. But don't worry about that you will get into it in later lessons.

For Loops

A while loop go through a list of items until something is true. A for loop goes through a list of items and can increment a variable each loop. Allowing you to do some for x number of times. Like print Hello World.

For Each Loops

A different way to do for loops mainly use for going through arrays. In a for each loop it goes through a list going item by item performing an action on the item then moving to the next one.
