Unit 3

Arrays and Algorithms

Overview

Unit 3 gets into better methods of storing and accessing data using Arrays and for loops.

1D arrays

An array is a way of storing variables of the same type neatly. Insted of having to change each variable by name you can just create an array of them and store, read, and change them in the array. An array in this case just discribes how the computer is storing the memory. Which is sequently, so each variable will be one after the other. The 1D simply refers to the dimension of the array. Which is the amount of arrays in arrays. But don't worry about that you will get into it in later lessons.

For Loops

While loops allow you to set a condition, and the loop will repeat as long as the condition is true. It is essentially an if statement that runs multiple times. For loops are very similar in that They keep looping, however you get to choose the number of times they repeat. This is useful because while you can do the same thing with while loops, for loops are much more efficient both to write and memory-wise. They work by having you declare a variable, declare an end condition, and determine what happens to that variable each loop For example, you declare an int i to be 0, say to run while i is less that 5, and to add 1 to i each loop. This would be written as "for(int i=0; i<5, i++)"

For Each Loops

For each loops are a different way of writing for loops. It is faster to write, and is generally used for going through arrays. A for each loop simply works by incrementing through an array and allowing you to access data from each item in it as it loops. It's basically the same as a for loop, just easier to write.
